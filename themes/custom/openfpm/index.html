The Introduction

OpenFPM is a scalable and open C++ framework for particles and mesh simulation

You can build scalable: Molecular dynamic,SPH, Granular flow,Finite differences, Particle Mesh codes and we provides examples on each of them. Despite our main interest in simulations the framework is highly documented and not limited to these tasks. The work is release under GPL 3.0
