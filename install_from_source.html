Installation from source



For Linux or OSX, clone our repo and to install in the default location use:
git clone https://github.com/IBirdSoft/openfpm_pdata_3.1.0.git && cd openfpm_pdata_3.1.0 && ./install && sudo make install

To choose where to install dependencies and library use:
git clone https://github.com/IBirdSoft/openfpm_pdata_3.1.0.git && cd openfpm_pdata_3.1.0 && ./install -i "/where/you/want/to/put/dependencies" -c "--prefix=/where/you/want/to/install" && make install
